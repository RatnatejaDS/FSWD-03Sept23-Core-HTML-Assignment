HTML Tags List

  1. <html>: The root element that encloses all content on the webpage.
  2. <head>: Contains meta-information about the document, such as the title and character encoding.
  3. <title>: Sets the title of the webpage displayed in the browser's tab.
  4. <body>: Contains the visible content of the webpage, including text, images, and links.
  5. <h1>: Represents a top-level heading with the largest font size.
  6. <ol>: Creates an ordered (numbered) list of items.
  7. <li>: Defines a list item within an ordered or unordered list.
  8. <a>: Creates hyperlinks to other web pages or resources.
